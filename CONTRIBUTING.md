# Contributing

Thanks for contributing to `input-mask`.

## Development

Install dependencies:

```bash
npm install
```

Run tests:

```bash
npm test
```

Build the package:

```bash
npm run build
```

## Guidelines

- Keep the library easy to use with HTML attributes only
- Preserve auto-initialization for CDN usage
- Add or update tests for behavior changes
- Keep documentation focused on practical usage

## Pull requests

Before opening a pull request:

1. run `npm test`
2. run `npm run build`
3. update the README if the public API or supported attributes change
